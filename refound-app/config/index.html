What the budget governs new
Universe denominator
3,400 / 247
Enumerate up to ~3,400 buyer queries and track presence on the top 247 destinations. A larger engagement widens both caps; a smaller one tightens them.
Effort ceiling per node
8 pts
Skip or defer nodes whose effort-to-win exceeds the value the budget implies per position. Keeps the inbox on high-return work, not vanity nodes.
Sprint velocity
6 / wk
Work items the team can ship per week at this tier — the rate the engine is allowed to release nodes into the inbox.
Team allocation
3 lanes
On-site SEO · Backlinks · UGC. Budget tier decides how much of each lane is staffed against this client.
Why this matters: the same engine running a $2K/mo client and a $20K/mo client should behave differently — smaller universe, slower release, tighter effort ceiling. Budget is the one dial that scales all of it.
