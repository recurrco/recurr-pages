Brief

Motion as structural, not decorative. The page breathes — numbers tick, timestamps live, rules reveal, cursors wait. Three categories: atmosphere (the technical substrate), instrumentation (live data in the margin), reveal (editorial pacing).

Notes

Each specimen is isolated, reusable, dependency-free (vanilla JS / Canvas / CSS). Motion always serves the content — atmosphere sits behind, instrumentation runs in the margin, reveals pace the read. Never a hero animation that blocks the headline.
